# Config Dashboard

`/config_dashboard` is the unified admin configuration interface. Requires `manage_guild` permission.

## Panels

### Staff Permissions
Map Discord roles to permission tiers (0-3). See [Staff Roles & Permissions](../getting-started/permissions.md) for the tier system.

### Scammer Auto-Ban
Toggle the auto-ban system on/off, configure the broadcast channel to listen to, set the rate limit between bans. See [Scammer Protection](../features/scammer-protection.md).

### Scammer Report
Configure the hall of shame channel and optional ping role for manual scammer reports. See [Scammer Protection](../features/scammer-protection.md).

### Ad Responses
Manage rule-based auto-responses for ad forum threads. See [Ad Responses](../features/ad-responses.md).

### Keep Alive
Add or remove forum channels from the keep-alive watch list. See [Keep Alive](../features/keep-alive.md).

### Auto-Publish
Add or remove announcement channels from the auto-publish watch list. See [Auto-Publish](../features/auto-publish.md).
