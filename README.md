# VideoEditingHubDocs

Staff documentation for the [Video Editing Hub](https://github.com/demigodmode/VideoEditingHubBot) Discord bot.

Built with [MkDocs](https://www.mkdocs.org/) and the [Material](https://squidfunk.github.io/mkdocs-material/) theme, hosted on [Read the Docs](https://readthedocs.org/).

## Local preview

```bash
pip install -r requirements.txt
mkdocs serve
```

Opens at `http://127.0.0.1:8000`. Changes auto-reload.
